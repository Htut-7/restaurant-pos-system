A modern Restaurant Point of Sale (POS) application built with React and Firebase. The system provides menu management, order processing, revenue tracking, and dashboard analytics for restaurant operations.

## Features

* Secure Authentication
* Dashboard Analytics
* Revenue Tracking
* Category Management
* Menu Management
* Point of Sale (POS)
* Cart Management
* Order Processing
* Quantity Controls
* Recent Orders Overview
* Responsive Design

## Tech Stack

**Frontend**

* React
* React Router
* CSS3
* Recharts
* React Icons

**Backend**

* Firebase Authentication
* Cloud Firestore

## Installation

```bash
git clone https://github.com/yourusername/restaurant-pos-system.git

cd restaurant-pos-system

npm install

npm run dev
```
